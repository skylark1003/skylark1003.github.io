【笔记】数值类型和字符（串）类型之间转换

Posted on 2022-08-09 Edited on 2022-08-16 In 笔记

一、类型转换

  • 数值转string：to_string(i)
  • char[]转数值：atoi(s)、atol(l)、atoll(ll)、atof(f)、atod(d)
  • string转数值： stoi(s)、stol(l)、stoll(ll)、stof(f)、stod(d)

二、大小写转换

  • 判断字符为小写字母、大写字母、数字：
    • 大写字母：ch >= 'A' && ch <= 'Z'
    • 小写字母：ch >= 'a' && ch <= 'z'
    • 数字：ch >= '0' && ch <= '9' 或 isdigit(char)
      （需要包含头文件#include <cctype>）
  • 大小写字母互转
    • 大写转小写：ch + 32
    • 小写转大写：ch - 32
