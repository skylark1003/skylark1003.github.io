371. Sum of Two Integers

Posted on 2022-09-06 In leetcode

一、描述

Given two integers a and b, return the sum of the two integers without using the operators + and -.

二、例子

Example 1:

Input: a = 1, b = 2
Output: 3

Example 2:

Input: a = 2, b = 3
Output: 5

Constraints:

  • -1000 <= a, b <= 1000

三、代码

解法来自Adityajha0808：

思路：

  1. 循环，直到第二个整数 “ b ”（反之亦然）变为0。
  2. 使用c变量记录每一位的进位情况。它被设置为无符号的，因为存在负数的情况，但进位始终为正。
  3. 使用异或运算XOR操作符（^）进行相加，0+1=1，1+0=1，0+0=0，1+1=0（含进位）。
  4. 产生的进位在高一位上生效，所以需要左移一位。b作为原本数字已经加过了，所以用其存储进位（即新的加数），直至没有新的进位产生。
